Podium

Podium is an E-Reader app for Music Scores, currently under active development. Highlights:
  • 100% browser-based.
    - nothing to install, no registration, no login.  Just use it!
    - developed for Google Chrome, but with luck (tm), runs on any modern desktop/tablet/smartphone browser
    - completely free, entirely open source, and wholly unsupported.
  • No proprietary formats: Podium is an alternative to generic PDF viewers, customized for the layout and annotation of scores in PDF format.
    - not a music library: works with PDF files stored on locally-accessible storage of on Google Drive cloud storage. There is no notion of "importing" in the app.
    - you might already have a collection of PDF scores: downloaded, scanned from paper copies, or created using music notation software. Podium is built to work with such scores.
    - uses PDF annotations and embedded document metadata to store user edits.
  • Supports touch screens, stylus, and mouse/keyboard input schemes.
  • Multiple layout schemes.
    - book-style, with realistic page flip animation
    - scrolling, both vertical or horizontal, with a user-specifiable number of pages visible at once
    - table, where all pages are potentially visible at once: useful for page layout tasks
  • Annotation tools.
    - free drawing tool
    - text annotation tool
    - rastrum tool for drawing staff lines
    - symbol tool, exposing all symbols from the included SMuFL-compliant Bravura music font
  • Tools for practice sessions.
    - metronome, including an animated conducting hand
    - on-screen piano keyboard, with functions for to assisted tuning
    - visualizer tool that continuously records the last 2 minutes (configurable) of a practice session, both audio and video, for instant playback
